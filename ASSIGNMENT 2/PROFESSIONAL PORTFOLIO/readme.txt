1. <!DOCTYPE html>: Defines the document type and version (HTML5).
2. <html lang="en">: Declares the language of the document.
3. <head>: Contains metadata about the webpage, such as character set, viewport settings, stylesheets, and title.
4. <meta charset="UTF-8">: Defines the character encoding (UTF-8).
5. <meta name="viewport" content="width=device-width, initial-scale=1.0">: Ensures the website is responsive across devices.
6. <link rel="stylesheet" href="...">`: Links external CSS stylesheets.
7. <title>: Defines the title of the web page shown in the browser tab.
8. <body>: Contains the visible content of the webpage.
9. <header>: Represents the header section of the page (contains navigation links and logo).
10. <nav>: Defines a set of navigation links.
11. <section>: Represents standalone sections like home & about & contact.
12. <h1>,<h2>,<h3>: Headers for content sections.
13. <p>: Paragraphs of text.
14. <button>: Clickable buttons used for actions (e.g., hire, experience).
15. <img src="">: Embeds images in the webpage.
16. <audio>: Embeds audio controls to play audio files.
17. <video>: Embeds video controls to play video files.
18. <form>: Defines an HTML form for user input in contact form.
19. <input>: Input fields within forms like text, email, number, etc..
20. <textarea>: Allows multiline text input in forms.
21. <ul>,<li>: Unordered lists for navigation and certificate gallery items.
22. <figure>: Groups images and captions in the gallery.
23. <table>: Displays tabular data for education details.
24. <footer>: Contains footer content, social media links, and copyright.
